# :clipboard: CRUD를 구현한 게시판

## 프로젝트 소개

 이 프로젝트는 Spring Boot를 공부하기 위해 개발한 게시판이다.
기본적으로 CRUD 기능이 구현되어 있고 AWS, Travis Ci, Nginx를 사용하여 무중단 배포까지 진행하였다.

## 프로젝트 기간

2021.03 ~ 2021.04


## 기술 스택
- FRONT-END: mustache, bootstrap
- BACK-END: spring boot
- DB: aws rds maria db
- ETC: github, travis ci, nginx
